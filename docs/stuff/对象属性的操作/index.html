前言

react hooks中最常用的useState，在每次对数据进行更新时，都需要对所有数据重新赋值。

  const [info, setInfo] = useState({name: 'rain', age: 18});  setInfo({      ...info,      name: 'realRain'  });

将这层逻辑封装一下，可以只更新需要增量修改部分，不用每次都写冗长的逻辑。

const useSetState = (initialState) => {  const [state, setState] = useState(initialState || {});
  const setMergeState = useCallback((patch) => {    setState((preState) => ({ ...preState, ...(isFunction(patch) ? patch(preState) : patch) }));  }, []);
  return [state, setMergeState];};

但是如果想要删去一个属性，需要如何操作呢？今天看到一种解构的解法，将对象中的某个属性置顶，质底，删除，用的挺活的。

删除某个属性#

const obj = {name: 'rain', age: 18, sex: 1};
// 删除sex属性const removeSexProp = ({sex, ...rest}) => rest// 删除固定属性方法const removeProp = (key) => ({[key]: _, ...rest}) => rest
const removeAge = removeProp('age');const removeName = removeProp('name');

将对象属性置顶#

const obj = {name: 'rain', age: 18, sex: 1};
const moveSex2Top = (object) => ({sex: undefined, ...object})

将某个对象沉底#

const obj = {name: 'rain', age: 18, sex: 1};
const moveSex2Bottom = ({name, ...rest}) => ({...rest, name})

一点一滴的积累吧。
